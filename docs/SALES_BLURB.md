# Sales Blurb — RepoRadar

RepoRadar is a zero-dependency Node.js CLI that turns the question "is this repo in good shape?" into a concrete, weighted score across seven dimensions -- tests, secret exposure, documentation, dependency hygiene, CI automation, git hygiene, and build/lint config -- and then goes one step further than any existing tool by generating a prioritized, agent-ready remediation plan you paste directly into Claude Code to fix the findings. The portfolio mode ranks every repo you own worst-first, giving an engineering manager or consultant an instant view of where the technical debt is concentrated. Built entirely on Node built-ins (no npm install, no external deps), it runs safely on untrusted code because it never executes a build. The combination of a rigorous scoring model, self-contained HTML dashboards with embedded charts, machine-readable JSON for CI integration, and an AI-native fix plan is not accidental -- it reflects the author's understanding that a health scanner is only valuable if it closes the loop from diagnosis to remediation, and that an AI coding agent is the fastest path to that loop.
